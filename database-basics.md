## Database Basics (Beginner Friendly)

### What is a Database?
A database is a place where websites store information in an organized way.
It is like a digital filing cabinet.

Examples:
- User details
- Blog posts
- Products

### What are Tables?
Tables store data inside a database.
They look like spreadsheets with rows and columns.

Examples:
- Users table
- Posts table

### What are Records (Rows)?
A record is one complete entry in a table.
Each row represents one item.

Example:
- One user
- One blog post

### What are Fields (Columns)?
Fields describe the details of a record.
Each column stores one type of information.

Examples:
- Name
- Email
- Title

### Simple Example: Blog Posts Table

Table name: posts

| Column Name | Description |
|------------|------------|
| id | Unique post ID |
| title | Blog post title |
| content | Blog post text |
| author | Author name |
| created_date | Date published |

Each row represents one blog post.
