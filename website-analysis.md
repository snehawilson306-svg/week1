## Overall Observations

### Common Components Across Websites
- Header and navigation appear on all websites
- Main content area is always the focus
- Footer contains legal, help, and contact information

### What Changes Between Websites
- Blogs focus on text and reading experience
- News websites show many headlines and images
- E-commerce websites focus on search and promotions
- Sidebars appear more on content-heavy sites

### Layout Patterns Observed
- Most websites follow a top-to-bottom structure
- Header → Navigation → Content → Footer is common
- Hero sections are common in e-commerce sites
- Visual hierarchy guides user attention
